
# BuildXSS
Build XSS is a Semi-Automatic, Open Source XSS Payload Generator.

**Language :** Bourne Again Shell (BASH)




## Download
*Open the Repository*
 - `git clone https://github.com/f4lcone/BuildXSS.git`

*Open BuildXSS Path*
- `cd BuildXSS`

## Usage
*Run with BASH*
- `./BuildXSS`
